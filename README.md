# CyberLab FIB

A cybersecurity learning project developed by a Computer Engineering student at the Universitat Politècnica de Catalunya.

## Objectives

- Improve Linux and networking fundamentals.
- Learn Git and GitHub.
- Study cybersecurity concepts and tools.
- Build a reproducible security laboratory.
- Automate security analysis tasks using Python and Bash.
- Document experiments, results and lessons learned.

## Planned technologies

- Linux
- Git and GitHub
- Python
- Bash
- Docker
- Wireshark
- Nmap
- OWASP Juice Shop
- Wazuh

## Ethical use

All experiments will be performed exclusively in isolated environments, intentionally vulnerable applications or authorized systems.

This project is intended for educational and defensive-security purposes.
